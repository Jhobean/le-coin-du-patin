Le coin du patin

Service d’aiguisage de patins pour les étudiants

🧊 Le service

Le coin du patin est un service offert aux étudiants pour l’aiguisage de leurs patins. La collecte se fait à l’école après les cours, et le retour est assuré le lendemain matin avant le début des classes. Le paiement se fait par virement Interac.

👤 Qui suis-je ?

Je suis Alexis, étudiant de secondaire 2. J’ai remarqué que transporter ses patins dans l’autobus pour les faire aiguiser est un vrai casse-tête pour plusieurs joueurs. Comme j’habite à proximité de l’école, j’ai décidé d’offrir ce service pour simplifier la vie de mes camarades.

⚙️ Comment ça fonctionne ?

  • Remplir le formulaire
  • Le glisser dans vos patins
  • Déposer les patins dans le bac prévu à cet effet
